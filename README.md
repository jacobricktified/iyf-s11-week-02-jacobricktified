## Project Description
This project demonstrates the creation of a reusable CSS design system for modern web pages. It includes a typography system, color palette using CSS custom properties, responsive navigation bar, card layout, footer, buttons, links, images, and overall page styling. The goal was to create a clean, responsive, and maintainable stylesheet that can be reused across multiple projects.

## Technologies Used
- HTML5
- CSS3
- CSS Variables (Custom Properties)
- Flexbox
- Media Queries
- Google Fonts (Poppins & Roboto)

## Features
- Responsive navigation bar
- Typography system with reusable font sizes
- CSS custom properties for colors and fonts
- Responsive card layout using Flexbox
- Styled buttons with hover effects
- Responsive footer with multiple columns
- Styled lists, links, and images
- Mobile-friendly layout using media queries

## Lessons Learned
While building this project, I learned how to organize CSS into logical sections to improve readability and maintainability. I also gained experience using CSS custom properties to create a consistent design system, Flexbox for responsive layouts, and media queries to ensure the website adapts well to different screen sizes.

## Challenges Faced
One challenge was ensuring the navigation bar and card layout remained responsive across different screen sizes. This was solved by using Flexbox together with media queries. Another challenge was maintaining consistent typography and colors throughout the stylesheet, which was addressed by defining reusable CSS variables in the `:root` selector.
